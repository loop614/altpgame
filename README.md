## Description
- [15 puzzle](https://en.wikipedia.org/wiki/15_puzzle)
- using 2D arrays
- only moves with 0 allowed

## Requirements
- cmake, build-essentials

## Quick Start
- cmake --build cmake-build-debug --target all -j 4
- ./cmake-build-debug/altpro
